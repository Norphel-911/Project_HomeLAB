# pfsense configuration:

==*explication*==
Une fois arrivé sur le consol de pfsense. on a uniquement le WAN(em0) qui est configuré. Le but c'est de créer un LAN premierement que les vms vont utiliser comme passerelle. 
on va assigner interface. 
  -on etiens pfsense, et ajoute une 2 une 2ème carte reseau. 
  -une fois avoir relancer la machine, on assign les interface dans option (1), emO = WAN(on le garde), em1 = LAN. 
  -on lui donne une IP LAN.
